# esGEO — Generative Engine Optimization

**esGEO** is a Spanish-language platform for **Generative Engine Optimization (GEO)** — the practice of structuring web content so it gets understood, trusted, and **cited by generative AI models** (ChatGPT, Perplexity, Claude, Gemini) rather than just ranked by search engines.

🔗 **Live:** https://esgeoai.vercel.app

## What it does

GEO shifts the goal from "rank on Google" to "be the source the AI quotes." esGEO teaches and tools a repeatable methodology to get there.

### Methodology (F1–F5, F6 in progress)

- **F1 · Fragmentation** — break content into self-contained, citable blocks
- **F2 · Semantic Hierarchy** — logical, machine-readable structure
- **F3 · Generative Authority** — build the credibility signals AI models weigh
- **F4 · Structured Data** — Schema.org tuned for LLM consumption
- **F5 · Technical Optimization** — performance and accessibility
- **F6 · Measurement** — citation tracking *(in production)*

### GEO features implemented

- Citation-oriented metadata and `Schema.org` structured data
- Semantic fragmentation and clean, meaningful URLs
- Speakable sections for voice synthesis
- Structured breadcrumbs and E-E-A-T authority signals

## Tech stack

React · TypeScript · Vite · Tailwind CSS · shadcn/ui · React Helmet · Schema.org
